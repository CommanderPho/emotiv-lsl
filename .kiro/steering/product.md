# Product Overview

Emotiv-LSL is an LSL (Lab Streaming Layer) server for Emotiv EPOC X EEG headsets. It enables real-time acquisition, streaming, and export of raw EEG and motion sensor data from Emotiv headsets via USB dongle.

## Key Features

- Real-time EEG data streaming (14 channels at 128Hz)
- Motion sensor data (accelerometer + gyroscope at 16Hz)
- Electrode quality monitoring
- LSL protocol for integration with neuroscience tools
- Cross-platform support (Windows, macOS, Linux)
- Docker deployment option
- MNE-Python integration for data export

## Target Users

Researchers and developers working with Emotiv EPOC X headsets who need raw data access for neuroscience experiments, BCI applications, or EEG analysis.
